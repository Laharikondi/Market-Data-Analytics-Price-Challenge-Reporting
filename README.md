**Market Pricing Data Analysis**

# Overview
This project collects market pricing data from different vendors, cleans and combines it using Python, and builds an interactive Power BI dashboard to analyze vendor performance, price variations, and trends.

# Project Structure
Market Pricing Data Analysis 
├── Data-sets # Contains all datasets (individual, combined, and transformed) 
├── Data-Extraction-Script # Python scripts for data extraction from APIs
├── ETL-Script # Scripts for data cleaning and ETL 
├── Resource # API keys and configuration files
└── Dash-board # Power BI dashboard

# Tools Used
Python – For data extraction and ETL
Power BI – For dashboard creation
APIs – Alpha Vantage, Yahoo Finance, Poly.io, Stock Data, Twelve Data

# Project Steps
Data Collection: Gather data from multiple APIs.
Pre-processing: Clean and combine datasets.
ETL: Calculate mean, median, price variation, and standard deviation.
Dashboard: Create visuals in Power BI to show vendor performance and trends.
Deployment: Publish or share the Power BI dashboard.

# Key Insights
Vendor-wise performance comparison
Price variation and outlier detection
Trend analysis across exchanges and price types
